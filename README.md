## 📖 Proje Hakkında

**MultiWayLearn**, İngilizce kelimeleri kalıcı olarak öğrenmek için
**6 Sefer Tekrar Prensibi (Spaced Repetition)** kullanan bir web uygulamasıdır.

Bir kelimeyi gerçekten öğrenmiş sayılmak için:
- 6 farklı zaman diliminde doğru cevaplanması gerekir
- Zaman dilimleri: **1. gün → 2. gün → 9. gün → 40. gün → 131. gün → 313. gün**
- 6. aşamayı geçen kelime "Öğrenildi" havuzuna taşınır

---

## 🚀 Özellikler

- ✅ Kullanıcı kayıt, giriş ve şifremi unuttum
- ✅ Kelime ekleme (resim + örnek cümle)
- ✅ 6 Sefer quiz algoritması (aralıklı tekrar)
- ✅ Kullanıcı ayarları (günlük yeni kelime sayısı)
- ✅ Analiz raporu
- ✅ Word Chain + LLM hikaye üretimi

---

## 🛠️ Teknoloji Yığını

| Katman | Teknoloji |
|--------|-----------|
| Frontend | React + Vite + Tailwind CSS |
| Backend | Node.js + Express |
| Veritabanı | SQLite (Prisma ORM) |
| Auth | JWT + bcrypt |
| LLM | Google Gemini API |
| Kod Kalitesi | SonarCloud |
